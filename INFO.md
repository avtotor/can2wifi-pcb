# INFO.md — назначение каждого файла проекта

Это **аппаратный проект KiCad**: печатная плата моста CAN→WiFi OBD-II
[can2wifi](https://github.com/avtotor/can2wifi) на ESP32-S3. Здесь описан каждый файл
репозитория: что это, зачем нужен и можно ли его удалять.

Принцип:
- **Source of truth** — данные, созданные вручную. Их удаление уничтожает работу,
  которую невозможно восстановить. **Удалять нельзя.**
- **Генерируемое / локальное** — то, что KiCad или тулчейн создаёт из source of truth,
  либо настройки конкретной машины. Удалять безопасно — пересоздаётся.

---

## Source of truth — УДАЛЯТЬ НЕЛЬЗЯ

Эти файлы определяют плату. Потеря любого = потеря невосстановимой работы.

| Файл | Что это | Зачем нужен |
|---|---|---|
| `can2wifi-pcb.kicad_sch` | **Схема.** Компоненты, цепи, номиналы, вся электрическая часть. | Электрический проект. Всё остальное (нетлист, BOM, связи на плате) выводится из неё. ERC-clean. |
| `can2wifi-pcb.kicad_pcb` | **Разводка платы.** Контур, размещение, трассировка меди, полигоны GND, переходные. | Физическая плата — именно это идёт в производство. Здесь часы размещения и трассировки; ниоткуда не пересоздаётся. |
| `can2wifi-pcb.kicad_pro` | **Файл проекта.** Design rules, net-классы, зазоры, стек слоёв, настройки плотов. | Связывает схему и плату, хранит правила, по которым проверяет DRC. Без него KiCad не откроет проект. |
| `can2wifi.pretty/OBD2_J1962_Male.kicad_mod` | **Кастомный футпринт** разъёма OBD-II J1962 (male). | Единственный футпринт не из стандартных библиотек KiCad. Плата на него ссылается; удаление ломает разъём J1962 на PCB. |
| `fp-lib-table` | **Таблица библиотек футпринтов.** Регистрирует локальную библиотеку `can2wifi.pretty`. | Без неё KiCad не найдёт кастомный футпринт J1962 выше. |

## Генерируемое / локальное — УДАЛЯТЬ БЕЗОПАСНО (пересоздаётся)

Ничего из этого нет в репозитории (в `.gitignore`), и большинства нет на диске — всё
производится из source of truth (или это настройки машины). Пересоздаётся из KiCad по
необходимости.

| Файл | Что это | Чем пересоздать |
|---|---|---|
| `doc/can2wifi-pcb-schematic.pdf` | PDF-плот схемы (просмотр без KiCad). | KiCad: плот схемы в PDF (или MCP `export_schematic_pdf`). |
| `doc/can2wifi-pcb-pcb.pdf` | PDF-плот платы (просмотр без KiCad). | KiCad: плот платы в PDF (или MCP `export_pcb_pdf`). |
| `can2wifi-pcb.net` | Нетлист, выгруженный из схемы. | KiCad: экспорт нетлиста (или MCP `export_netlist`). |
| `fab/can2wifi-pcb-gerbers.zip` | Пакет gerber + drill для производства (JLCPCB). | KiCad plot / MCP `export_gerbers` + `export_drill`. |
| `fab/can2wifi-pcb-bom-jlcpcb.csv` | Bill of Materials в формате JLCPCB. | MCP `export_bom` (исключить DNP: D5, R8). |
| `fab/can2wifi-pcb-cpl-jlcpcb.csv` | Размещение компонентов (pick-and-place) для PCBA. | MCP `export_pos` (CPL). |
| `can2wifi-pcb.kicad_prl` | Локальное состояние проекта KiCad (панели, последний вид, локальные настройки). | KiCad пересоздаёт при открытии. Привязано к машине. |
| `.claude/settings.local.json` | Локальные настройки/разрешения Claude Code для этой машины. | К плате отношения не имеет; специфично для окружения. |

> Весь каталог `fab/` — это генерируемый вывод: **развести плату → DRC-clean →
> экспортировать fab-файлы**. Это поставка на производство, а не source.
